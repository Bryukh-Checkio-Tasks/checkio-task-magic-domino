Build square matrix size✕size of integers, where sum of numbers in each row, column and diagonal equals number. This matrix should be builded from unique, vertically oriented classic domino tiles (from 0 to 6 inclusive on each half of tile, also named double-six set).

Here is example for size = 4 and number = 5 

     5  5 5 5 5  5
        _______ 
     5 |0 0 2 3
     5 |0 4 1 0

     5 |4 0 0 1
     5 |1 1 2 1

You get size and number as input to your checkio function and should return matrix (list/tuple of rows (list/tuple)). For above example: ((0, 0, 2, 3), (0, 4, 1, 0), ...)

Precondition:
size in (4, 6)
All input cases are solvable.

Hint:
Your solution should run nearly half a minute on your local pc for thoughest inputs, so don't be afraid.
