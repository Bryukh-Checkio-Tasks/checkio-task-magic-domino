Build square matrix w✕w of integers, where sum of numbers in each row, column and diagonal equals s. This matrix should be builded from unique, vertically oriented classic domino tiles (from 0 to 6 inclusive on each half of tile, also named double-six set).

Here is example for w = 4 and s = 5 

     5  5 5 5 5  5
        _______ 
     5 |0 0 2 3
     5 |0 4 1 0

     5 |4 0 0 1
     5 |1 1 2 1

You get w and s (width/height of matrix and sum value) to your checkio function

Precondition:
w in (4, 6)
All input values are solvable.

Hint:
Your solution should run nearly half a minute on your local pc for thoughest inputs, so don't be afraid.
