Build square matrix w✕w of integers, where sum of all numbers in each row, column and diagonal equals s. This matrix should be builded from classic domino tiles (from 0 to 6 inclusive on each part of bone, also named double-six set).

You get w and s (width/height of matrix and sum value) to your checkio function

Precondition:
4 <= w <= 6
All input values are correct, and matrix can be builded.
