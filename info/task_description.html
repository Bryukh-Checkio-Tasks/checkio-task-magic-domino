Build square matrix w✕w of integers, where sum of all numbers in each row, column and diagonal equals s. This matrix should be builded from vertically oriented classic domino tiles (from 0 to 6 inclusive on each half of tile, also named double-six set).

You get w and s (width/height of matrix and sum value) to your checkio function

Precondition:
w in (4, 6)
All input values are solvable.
